05 UX RULES
01Create session → 바로 세션 화면 진입, 추가 뎁스 없음. 날짜·시간은 자동 기록되어 제목 기본값이 됨
02컨트롤은 레일의 같은 자리에서 상태 전환: Ready(Start + End 딤) → Listening(Pause / End) → Paused(Resume / End)
03타깃 언어 기본 Korean, 원문은 자동 감지 표기(● ENGLISH — AUTO). 소스는 Microphone / Tab audio 중 하나만 활성
04전사는 좌 원문(산세리프 14.5) / 우 번역(세리프 16), 세그먼트마다 타임스탬프. 라이브 캐럿은 번역 컬럼 끝에만
05자동 저장 상시 + .TXT 다운로드 / 클립보드 복사. 오토스크롤은 하단 고정, 새 줄은 아래에 추가
06솔리드 레드는 전진 동작(Start/Resume) 전용, 위험 동작(End)은 아웃라인 레드 → 호버 시 채움
